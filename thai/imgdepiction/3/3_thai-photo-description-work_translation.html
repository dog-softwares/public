タイ語解説：日本の企業で働く

ページタイトル

タイ語 แบบฝึกหัดการอธิบายภาพ: การทำงานในบริษัทญี่ปุ่น
発音 bɛ̀ɛp-fʉ̀k-hàt gaan-à-tí-baai pâap: gaan-tam-ngaan nai bɔɔ-rí-sàt yîi-bpùn
和訳 画像描写トレーニング：日本の会社で働くこと

講師へのアドバイス

タイ語 คำแนะนำสำหรับครูผู้สอน:
発音 kam-nɛ́-nam sǎm-ràp kruu-pûu-sɔ̌ɔn
和訳 講師へのアドバイス：
1.
タイ語นี่คือแบบฝึกหัดการอธิบายภาพและการฝึกพูดตาม (Shadowing)
発音nîi kʉʉ bɛ̀ɛp-fʉ̀k-hàt gaan-à-tí-baai pâap lɛ́ gaan-fʉ̀k pûut dtaam (Shadowing)
和訳これは画像描写とシャドーイングの練習です。
2.
タイ語ให้คุณครูสร้างประโยคภาษาไทย (พูดเป็นภาษาไทย) เกี่ยวกับสิ่งที่จินตนาการได้จากภาพแต่ละภาพ
発音hâi kun-kruu sâang bprà-yòok paa-sǎa tai (pûut bpen paa-sǎa tai) gìao-gàp sìng tîi jin-dtà-naa-gaan dâai jàak pâap dtɛ̀ɛ-lá pâap
和訳講師は、各画像からイメージできる事柄についてタイ語の文章を作成してください（タイ語で話してください）。
3.
タイ語ให้นักเรียนพูดตามประโยคภาษาไทยที่คุณครูพูดให้เหมือนที่สุด
発音hâi nák-rian pûut dtaam bprà-yòok paa-sǎa tai tîi kun-kruu pûut hâi mʉ̌an tîi-sùt
和訳生徒は、講師が話したタイ語をできるだけそっくり真似して話してください。
4.
タイ語ให้คุณครูช่วยแนะนำเรื่องการออกเสียงและคำศัพท์ใหม่ๆ
発音hâi kun-kruu chûai nɛ́-nam rʉ̂ang gaan-ɔ̀ɔk-sǐang lɛ́ kam-sàp mài-mài
和訳講師は発音や新しい単語について指導してください。
5.
タイ語สำหรับคำศัพท์ใหม่ๆ ให้พิมพ์ส่งในช่องแชท
発音sǎm-ràp kam-sàp mài-mài hâi pim sòng nai chɔ̂ng-chɛ́t
和訳新しい単語はチャット欄にタイプして送ってください。
6.
タイ語หลังจากจบบทเรียน ให้นักเรียนสร้างรายการคำศัพท์ของตนเอง
発音lǎng-jàak jòp bòt-rian hâi nák-rian sâang raai-gaan kam-sàp kɔ̌ɔng dton-eeng
和訳レッスン終了後、生徒は自分用の単語リストを作成してください。
7.
タイ語คำศัพท์เหล่านั้นจะใช้ทบทวนในบทเรียนครั้งถัดไป
発音kam-sàp lào-nán jà chái tóp-tuan nai bòt-rian kráng tàt-bpai
和訳それらの単語は次回のレッスンで復習します。
8.
タイ語ในบทเรียนครั้งถัดไป ให้นักเรียนพยายามพูดด้วยตนเองให้มากที่สุด
発音nai bòt-rian kráng tàt-bpai hâi nák-rian pá-yaa-yaam pûut dûai dton-eeng hâi mâak tîi-sùt
和訳次回のレッスンでは、生徒はできるだけ自分で話すように努めてください。

各画像のタイトル

1. プレッシャーに追われる会社員
タイ語พนักงานบริษัทที่ถูกกดดันด้วยความกดดัน
発音pá-nák-ngaan bɔɔ-rí-sàt tîi tùuk gòt-dan dûai kwaam-gòt-dan
和訳プレッシャーによって圧迫されている会社員
2. 時間に追われるビジネスマン
タイ語นักธุรกิจที่ถูกกดดันด้วยเวลา
発音nák-tú-rá-gìt tîi tùuk gòt-dan dûai wee-laa
和訳時間に圧迫されているビジネスマン
3. ストレスで沈む人
タイ語คนที่จมอยู่กับความเครียด
発音kon tîi jom yùu gàp kwaam-krîat
和訳ストレスに沈んでいる人
4. ストレス解消に依存する人
タイ語คนที่พึ่งพาการคลายเครียด
発音kon tîi pʉ̂ng-paa gaan-klaai-krîat
和訳ストレス解消に頼っている（依存している）人
5. 仕事依存症のビジネスマン
タイ語นักธุรกิจที่ติดงาน
発音nák-tú-rá-gìt tîi dtìt ngaan
和訳仕事中毒の（仕事にハマっている）ビジネスマン
