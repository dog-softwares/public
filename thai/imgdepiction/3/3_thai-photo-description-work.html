แบบฝึกหัดการอธิบายภาพ: การทำงานในบริษัทญี่ปุ่น

คำแนะนำสำหรับครูผู้สอน:

  1. นี่คือแบบฝึกหัดการอธิบายภาพและการฝึกพูดตาม (Shadowing)
  2. ให้คุณครูสร้างประโยคภาษาไทย (พูดเป็นภาษาไทย) เกี่ยวกับสิ่งที่จินตนาการได้จากภาพแต่ละภาพ
  3. ให้นักเรียนพูดตามประโยคภาษาไทยที่คุณครูพูดให้เหมือนที่สุด
  4. ให้คุณครูช่วยแนะนำเรื่องการออกเสียงและคำศัพท์ใหม่ๆ
  5. สำหรับคำศัพท์ใหม่ๆ ให้พิมพ์ส่งในช่องแชท
  6. หลังจากจบบทเรียน ให้นักเรียนสร้างรายการคำศัพท์ของตนเอง
  7. คำศัพท์เหล่านั้นจะใช้ทบทวนในบทเรียนครั้งถัดไป
  8. ในบทเรียนครั้งถัดไป ให้นักเรียนพยายามพูดด้วยตนเองให้มากที่สุด

1. พนักงานบริษัทที่ถูกกดดันด้วยความกดดัน

2. นักธุรกิจที่ถูกกดดันด้วยเวลา

3. คนที่จมอยู่กับความเครียด

4. คนที่พึ่งพาการคลายเครียด

5. นักธุรกิจที่ติดงาน
